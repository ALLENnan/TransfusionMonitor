# TransfusionMonitor
####项目简介
基于微处理的智能输液监控系统(广外创新实验室团队)，一款智能化的医疗设备，通过与点滴硬件端的连接，实现对输液的监控和控制；可以实时监控输液速度，智能调整输液速度以及输液结束前的智能提醒
